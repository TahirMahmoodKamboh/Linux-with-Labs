# Labs: Package Management in Linux

## Lab 1: Update Package List
- Update the package list on your system.
  ```bash
  # For Debian/Ubuntu
  sudo apt update
  # For Fedora/RHEL/CentOS
  sudo dnf check-update
  # For openSUSE
  sudo zypper refresh
  # For Arch/Manjaro
  sudo pacman -Syu
  ```
**Solution:**
- The command completes without errors and updates the package database.

## Lab 2: Install a Package
- Install the `htop` system monitor tool.
  ```bash
  # For Debian/Ubuntu
  sudo apt install htop
  # For Fedora/RHEL/CentOS
  sudo dnf install htop
  # For openSUSE
  sudo zypper install htop
  # For Arch/Manjaro
  sudo pacman -S htop
  ```
**Solution:**
- Run `htop` to verify installation. The htop interface should appear.

## Lab 3: Remove a Package
- Remove the `htop` package from your system.
  ```bash
  # For Debian/Ubuntu
  sudo apt remove htop
  # For Fedora/RHEL/CentOS
  sudo dnf remove htop
  # For openSUSE
  sudo zypper remove htop
  # For Arch/Manjaro
  sudo pacman -R htop
  ```
**Solution:**
- Running `htop` should now result in a "command not found" error.

## Lab 4: Search for a Package
- Search for the `curl` package in your distribution's repositories.
  ```bash
  # For Debian/Ubuntu
  apt search curl
  # For Fedora/RHEL/CentOS
  dnf search curl
  # For openSUSE
  zypper search curl
  # For Arch/Manjaro
  pacman -Ss curl
  ```
**Solution:**
- The output should list the `curl` package and a brief description.

## Lab 5: Show Package Information
- Display detailed information about the `curl` package.
  ```bash
  # For Debian/Ubuntu
  apt show curl
  # For Fedora/RHEL/CentOS
  dnf info curl
  # For openSUSE
  zypper info curl
  # For Arch/Manjaro
  pacman -Qi curl
  ```
**Solution:**
- The output should show version, description, dependencies, and more about `curl`.

## Lab 6: Clean Up Unused Packages
- Remove unused packages and clean the package cache.
  ```bash
  # For Debian/Ubuntu
  sudo apt autoremove
  sudo apt clean
  # For Fedora/RHEL/CentOS
  sudo dnf autoremove
  sudo dnf clean all
  # For openSUSE
  sudo zypper clean --all
  # For Arch/Manjaro
  sudo pacman -Rns $(pacman -Qdtq)
  sudo pacman -Sc
  ```
**Solution:**
- The commands should complete without errors and free up disk space.
