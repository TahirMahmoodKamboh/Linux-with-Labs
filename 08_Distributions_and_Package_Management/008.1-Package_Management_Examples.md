# Package Management Examples

## apt (Debian/Ubuntu)

### What is apt?
- `apt` is the package manager for Debian-based systems (like Ubuntu).
- It handles installing, updating, upgrading, and removing software packages.

### Common apt Commands
- Update package list:
  ```bash
  sudo apt update
  ```
- Upgrade all installed packages:
  ```bash
  sudo apt upgrade
  ```
- Install a package:
  ```bash
  sudo apt install package_name
  ```
- Remove a package:
  ```bash
  sudo apt remove package_name
  ```
- Search for a package:
  ```bash
  apt search package_name
  ```
- Show package details:
  ```bash
  apt show package_name
  ```
- Clean up unused packages:
  ```bash
  sudo apt autoremove
  sudo apt clean
  ```

#### Example: Install and Remove curl
```bash
sudo apt update
sudo apt install curl
curl --version
sudo apt remove curl
```

## dnf (Fedora/RHEL 8+)

### What is dnf?
- `dnf` is the modern package manager for Fedora and newer Red Hat-based systems (RHEL 8+, CentOS 8+).

### Common dnf Commands

- Install a package:

- Search for a package:

- Show package details:
  ```bash
  dnf info package_name
  ```

- Clean up unused packages:

## pacman (Arch Linux/Manjaro)

### What is pacman?
- `pacman` is the package manager for Arch Linux and Manjaro systems.
- It handles installing, updating, upgrading, and removing software packages.

### Common pacman Commands
- Update package list and upgrade all packages:
  ```bash
  sudo pacman -Syu
  ```
- Install a package:
  ```bash
  sudo pacman -S package_name
  ```
- Remove a package:
  ```bash
  sudo pacman -R package_name
  ```
- Search for a package:
  ```bash
  pacman -Ss package_name
  ```
- Show package details:
  ```bash
  pacman -Qi package_name
  ```
- Clean up unused packages:
  ```bash
  sudo pacman -Rns $(pacman -Qdtq)
  sudo pacman -Sc
  ```

#### Example: Install and Remove htop
```bash
sudo pacman -Syu
sudo pacman -S htop
htop --version
sudo pacman -R htop
```
