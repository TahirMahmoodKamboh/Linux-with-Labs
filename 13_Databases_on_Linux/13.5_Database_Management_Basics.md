# Database Management Basics

## Common Tasks
- Creating databases and users
- Granting privileges
- Connecting to the database

## MySQL/MariaDB Example
```bash
sudo mysql -e "CREATE DATABASE testdb;"
sudo mysql -e "CREATE USER 'testuser'@'localhost' IDENTIFIED BY 'password';"
sudo mysql -e "GRANT ALL PRIVILEGES ON testdb.* TO 'testuser'@'localhost';"
```

## PostgreSQL Example
```bash
sudo -u postgres createdb testdb
sudo -u postgres createuser testuser
sudo -u postgres psql -c "ALTER USER testuser WITH PASSWORD 'password';"
sudo -u postgres psql -c "GRANT ALL PRIVILEGES ON DATABASE testdb TO testuser;"
```
