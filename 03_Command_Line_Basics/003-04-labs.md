## Lab 1: Using man Pages
- Use `man` to read the manual for a command.
  ```bash
  $ man pwd
  # (Manual page opens, press q to quit)
  ```

## Lab 2: Using --help
- Use `--help` to get a quick summary of a command's options.
  ```bash
  $ pwd --help
  Usage: pwd [OPTION]...
  Print the full filename of the current working directory.
  ...
  ```

## Lab 3: Using whatis
- Use `whatis` to get a one-line description of a command.
  ```bash
  $ whatis pwd
  pwd (1) - print name of current/working directory
  $ whatis mkdir
  mkdir (1) - make directories
  ```

---

Practice these help commands to become self-sufficient on the command line!
