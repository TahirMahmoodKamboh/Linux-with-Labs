## Lab 1: Who Are You?
- Run `whoami` and `id` to see your username and user details.
  ```bash
  $ whoami
  student
  $ id
  uid=1000(student) gid=1000(student) groups=1000(student),27(sudo)
  ```

## Lab 2: System Info
- Use `uname -a` and `hostname` to check system details.
  ```bash
  $ uname -a
  Linux mypc 5.15.0-50-generic #56-Ubuntu SMP ...
  $ hostname
  mypc
  ```

## Lab 3: Navigation
- Practice `pwd`, `ls`, `cd`, and `ls -l`.
  ```bash
  $ pwd
  /home/student
  $ ls
  Documents  Downloads
  $ cd Documents
  $ ls -l
  total 0
  ```

## Lab 4: File Operations
- Create, move, and remove files and directories.
  ```bash
  $ mkdir testlab
  $ cd testlab
  $ touch file1.txt file2.txt
  $ ls
  file1.txt  file2.txt
  $ mv file1.txt fileA.txt
  $ rm file2.txt
  $ cd ..
  $ rm -r testlab
  ```

## Lab 5: Getting Help
- Use `man`, `--help`, and `whatis`.
  ```bash
  $ man ls
  $ ls --help
  $ whatis ls
  ls (1) - list directory contents
  ```
