# Shell Scripting: Script Arguments and Exit Status

## Script Arguments
- $0: Script name
- $1, $2, ...: Positional arguments
- $@: All arguments
- $# : Number of arguments

## Example: Argument Handling
```bash
if [ $# -lt 1 ]; then
  echo "Usage: $0 arg1 [arg2 ...]"
  exit 1
fi
echo "First argument: $1"
```

## Exit Status
- 0: Success
- Non-zero: Error

## Example: Checking Exit Status
```bash
ls /notfound
if [ $? -ne 0 ]; then
  echo "Command failed!"
fi
```

## Lab
1. Write a script that checks if at least two arguments are provided.
2. Write a script that runs a command and prints if it succeeded or failed.

---

## Solutions
1.
```bash
#!/bin/bash
if [ $# -lt 2 ]; then
  echo "Need two arguments."
  exit 1
fi
echo "Arguments: $1 $2"
```
2.
```bash
#!/bin/bash
ls $1
if [ $? -eq 0 ]; then
  echo "Success"
else
  echo "Failure"
fi
```
