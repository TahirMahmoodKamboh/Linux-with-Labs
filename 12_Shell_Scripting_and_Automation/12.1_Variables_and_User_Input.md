## User Arguments
Shell scripts can accept arguments from the command line:

### Example: Accessing Arguments
```bash
#!/bin/bash
echo "Script name: $0"
echo "First argument: $1"
echo "Second argument: $2"
echo "All arguments: $@"
```

### Example: Sum Two Numbers from Arguments
```bash
#!/bin/bash
sum=$(($1 + $2))
echo "Sum: $sum"
```

### Example: Check Number of Arguments
```bash
#!/bin/bash
if [ $# -lt 2 ]; then
	echo "Usage: $0 num1 num2"
	exit 1
fi
echo "You entered: $1 and $2"
```

---

## More Examples

### Exporting and Using Environment Variables
```bash
#!/bin/bash
export MY_VAR="Hello from env!"
echo $MY_VAR
```

### Concatenating Strings
```bash
#!/bin/bash
first="Linux"
second="Rocks"
message="$first $second!"
echo $message
```

### Default Value for User Input
```bash
#!/bin/bash
read -p "Enter your city [Default: London]: " city
city=${city:-London}
echo "City: $city"
```
