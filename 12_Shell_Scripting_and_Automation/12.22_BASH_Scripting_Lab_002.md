# Case Statement

**`case` statement** in Bash (and POSIX `sh`) is like a **more readable alternative to multiple `if-elif-else` conditions**, especially when you want to compare **one variable against multiple possible values**.

Itâ€™s **ideal for menus, commands, options, or pattern matching**.

---

## ğŸ§  Syntax of `case`

```bash
case <variable> in
    pattern1)
        # commands if variable matches pattern1
        ;;
    pattern2|pattern3)
        # commands if variable matches pattern2 OR pattern3
        ;;
    *)
        # default commands (like else)
        ;;
esac
```

* `variable` â†’ value you want to check
* `pattern` â†’ can include wildcards like `*`, `?`
* `;;` â†’ marks **end of that block**
* `*` â†’ **default case**
