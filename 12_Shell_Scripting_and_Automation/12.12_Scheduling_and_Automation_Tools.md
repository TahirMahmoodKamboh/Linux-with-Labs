# Shell Scripting: Scheduling and Automation Tools

## cron
- Schedule recurring tasks
- Edit with crontab -e

### Example: Run script every day at 2am
```
0 2 * * * /path/to/script.sh
```

## at
- Schedule one-time tasks
- Example:
```bash
echo "echo Hello" | at now + 1 minute
```

## anacron
- Like cron, but for systems not always running

---

## Lab
1. Schedule a script to run every hour using cron.
2. Use at to schedule a message for 5 minutes from now.

## Solutions
1. Add this to crontab:
```
0 * * * * /path/to/script.sh
```
2.
