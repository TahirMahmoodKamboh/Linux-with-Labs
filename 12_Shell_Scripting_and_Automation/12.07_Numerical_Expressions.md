# Shell Scripting: Numerical Expressions

Shell scripts can perform arithmetic operations using several methods.

## Arithmetic with $(( ))
```bash
a=5
b=3
sum=$((a + b))
echo "Sum: $sum"
```

## Using let
```bash
let result=5*3
echo $result
```

## Using expr
```bash
a=10
b=2
result=$(expr $a / $b)
echo "Quotient: $result"
```

## Increment/Decrement
```bash
count=0
((count++))
echo $count
```

---

## Lab
1. Write a script that multiplies two numbers given as arguments.
2. Create a script that checks if a number is even or odd.

---

## Solutions
1.
```bash
#!/bin/bash
result=$(($1 * $2))
echo "Product: $result"
```
2.
```bash
#!/bin/bash
if [ $(($1 % 2)) -eq 0 ]; then
  echo "Even"
else
  echo "Odd"
fi
```
