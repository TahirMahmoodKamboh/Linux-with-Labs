# Shell Scripting: Arrays and Associative Arrays

## Indexed Arrays
```bash
fruits=(apple banana cherry)
echo ${fruits[1]} # banana
echo ${fruits[@]} # all elements
```

## Looping Through Arrays
```bash
for fruit in "${fruits[@]}"; do
  echo $fruit
done
```

## Associative Arrays (Bash 4+)
```bash
declare -A capitals
capitals[France]=Paris
capitals[Germany]=Berlin
echo ${capitals[France]}
```

---

## Lab
1. Create an array of numbers and print their sum.
2. Use an associative array to store and print country-capital pairs.

---

## Solutions
1.
```bash
#!/bin/bash
nums=(1 2 3 4 5)
sum=0
for n in "${nums[@]}"; do
  sum=$((sum + n))
done
echo $sum
```
2.
```bash
#!/bin/bash
declare -A capitals
capitals[USA]=Washington
capitals[UK]=London
for country in "${!capitals[@]}"; do
  echo "$country: ${capitals[$country]}"
done
```
