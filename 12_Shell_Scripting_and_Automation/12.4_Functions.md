## Lab: Practice with Functions

1. Write a function that adds two numbers.
2. Create a script with a function that greets a user by name.

---

## Solutions

1. 
```bash
add() {
  echo $(($1 + $2))
}
add 3 5
```

2. 
```bash
greet() {
  echo "Hello, $1!"
}
greet "Alice"
```
