# Shell Scripting: Debugging and Error Handling

## Debugging Scripts
- Use set -x to trace execution
- Use set -e to exit on error

## Example: Enable Debugging
```bash
set -x
echo "Debugging enabled"
set +x
echo "Debugging off"
```

## Error Handling
- Check exit status ($?)
- Use traps for cleanup

## Example: Trap
```bash
trap 'echo "Script interrupted!"; exit 1' INT
```

## Lab
1. Write a script that exits if any command fails.
2. Add a trap to handle script interruption (Ctrl+C).

---

## Solutions
1.
```bash
#!/bin/bash
set -e
cp notfound.txt nowhere.txt
echo "This will not print if cp fails."
```
2.
```bash
#!/bin/bash
trap 'echo "Interrupted!"; exit 1' INT
while true; do
  sleep 1
done
```
