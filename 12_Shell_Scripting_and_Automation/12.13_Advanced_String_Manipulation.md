# Shell Scripting: Advanced String Manipulation

## String Length
```bash
str="hello"
echo ${#str}
```

## Substring Extraction
```bash
echo ${str:1:3} # ell
```

## Pattern Replacement
```bash
str="foo bar foo"
echo ${str/foo/bar} # first only
echo ${str//foo/bar} # all
```

## Parameter Expansion
```bash
name=${USER:-guest}
```

---

## Lab
1. Extract the domain from an email address.
2. Replace all spaces in a string with underscores.

---

## Solutions
1.
```bash
#!/bin/bash
email="user@example.com"
domain=${email#*@}
echo $domain
```
2.
```bash
#!/bin/bash
str="hello world"
echo ${str// /_}
```
