# Assigning IP Addresses on Ubuntu and Red Hat

This guide explains how to assign static IP addresses on Ubuntu (using Netplan) and Red Hat (using network scripts or nmcli).

---

## Ubuntu (Netplan)

### Steps:
1. Edit the Netplan configuration file (usually `/etc/netplan/01-netcfg.yaml` or similar):

```yaml
network:
  version: 2
  ethernets:
    eth0:
      dhcp4: no
      addresses: [192.168.1.100/24]
      gateway4: 192.168.1.1
      nameservers:
        addresses: [8.8.8.8,8.8.4.4]
```

2. Apply the configuration:
```bash
sudo netplan apply
```

---

## Red Hat (ifcfg or nmcli)

### Using ifcfg scripts (RHEL 7 and earlier):
1. Edit `/etc/sysconfig/network-scripts/ifcfg-eth0`:
```
DEVICE=eth0
BOOTPROTO=none
ONBOOT=yes
IPADDR=192.168.1.100
NETMASK=255.255.255.0
GATEWAY=192.168.1.1
DNS1=8.8.8.8
```
2. Restart the network service:
```bash
sudo systemctl restart network
```

### Using nmcli (RHEL 7+ and 8+):
```bash
sudo nmcli con mod eth0 ipv4.addresses 192.168.1.100/24
sudo nmcli con mod eth0 ipv4.gateway 192.168.1.1
sudo nmcli con mod eth0 ipv4.dns "8.8.8.8 8.8.4.4"
sudo nmcli con mod eth0 ipv4.method manual
sudo nmcli con up eth0
```
