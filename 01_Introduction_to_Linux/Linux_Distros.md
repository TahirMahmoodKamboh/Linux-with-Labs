1.7 Linux Distributions (Distros)

A Linux distribution is a packaged operating system that includes:

Linux kernel

Package manager

Desktop environment (optional)

System utilities

Popular Linux Distributions

| Distribution | Use Case                    |
| ------------ | --------------------------- |
| Ubuntu       | Beginners, Servers, Cloud   |
| Debian       | Stability-focused systems   |
| CentOS       | Enterprise servers (legacy) |
| Rocky Linux  | RHEL replacement            |
| AlmaLinux    | Enterprise-grade Linux      |
| Fedora       | Latest features             |
| Arch Linux   | Advanced users              |
