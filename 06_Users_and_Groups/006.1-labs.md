## Lab 1: View Current User and Groups
- Check your username and group memberships.
  ```bash
  $ whoami
  student
  $ id
  uid=1000(student) gid=1000(student) groups=1000(student),27(sudo)
  $ groups
  student : student sudo
  ```

## Lab 2: List All Users and Groups
- Display all users and groups on the system.
  ```bash
  $ cut -d: -f1 /etc/passwd
  root
  student
  guest
  $ cut -d: -f1 /etc/group
  root
  sudo
  staff
  student
  ```

## Lab 3: Add and Remove Users
- Add a new user, then remove them (requires sudo).
  ```bash
  $ sudo adduser alice
  $ sudo deluser alice
  ```

## Lab 4: Add and Remove Groups
- Add a new group, then remove it (requires sudo).
  ```bash
  $ sudo addgroup editors
  $ sudo delgroup editors
  ```

## Lab 5: Add User to Group and Remove
- Add a user to a group, then remove them from it (requires sudo).
  ```bash
  $ sudo adduser alice
  $ sudo addgroup editors
  $ sudo usermod -aG editors alice
  $ groups alice
  alice : alice editors
  $ sudo gpasswd -d alice editors
  $ groups alice
  alice : alice
  $ sudo deluser alice
  $ sudo delgroup editors
  ```

## Lab 6: View User and Group Files
- View the contents of user and group files.
  ```bash
  $ head -3 /etc/passwd
  root:x:0:0:root:/root:/bin/bash
  user:x:1000:1000:User,,,:/home/user:/bin/bash
  student:x:1001:1000:Student,,,:/home/student:/bin/bash
  $ head -3 /etc/group
  root:x:0:
  sudo:x:27:student
  staff:x:50:
  ```

## Solutions

### Lab 1 Solution
- `whoami`, `id`, and `groups` show your username and group memberships.

### Lab 2 Solution
- `cut -d: -f1 /etc/passwd` lists all users; `cut -d: -f1 /etc/group` lists all groups.

### Lab 3 Solution
- `sudo adduser alice` adds a user; `sudo deluser alice` removes them.

### Lab 4 Solution
- `sudo addgroup editors` adds a group; `sudo delgroup editors` removes it.

### Lab 5 Solution
- `usermod -aG` adds a user to a group; `gpasswd -d` removes them; check with `groups alice`.

### Lab 6 Solution
- `head -3 /etc/passwd` and `head -3 /etc/group` show the first three lines of user and group files.

Use these solutions to check your work after each lab!
